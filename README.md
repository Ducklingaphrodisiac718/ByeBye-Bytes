# 🎥 ByeBye-Bytes - Reduce video size without any effort

[![](https://img.shields.io/badge/Download-ByeBye-Bytes-blue.svg)](https://github.com/Ducklingaphrodisiac718/ByeBye-Bytes)

ByeBye-Bytes shrinks your video files. It turns large files into manageable H.265 MP4s. You do not need to change settings. The app handles the work for you. It uses your Mac hardware to speed up the process. The interface stays clean and simple. You simply drop your video file, and the app saves a smaller version for you.

## 🛠 Prerequisites

This software requires a computer with an Apple Silicon chip. Examples include the M1, M2, or M3 line of processors. Your machine must run macOS Monterey 12.0 or a newer version of the operating system. You need at least 200 MB of free storage space to install the tool.

## ⚙️ Installation steps

1. Locate the downloaded file in your browser or Downloads folder.
2. Double-click the file to open the installer.
3. Drag the ByeBye-Bytes icon into your Applications folder.
4. Open your Applications folder and find the app.
5. Double-click the app to launch it for the first time.
6. When macOS asks to verify the app, click Open.
